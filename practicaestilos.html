font-size font-size: 20px; Cambia el tamaño del texto a 20px.
background background: yellow; El fondo será amarillo.
border border: 2px blue; Añade un borde negro de 2px alrededor del elemento.
margin margin: 20px; Añade un margen de 20px alrededor del elemento.
padding padding: 10px; Añade un padding de 10px dentro del elemento.
text-align text-align: center; Alinea el texto al centro del elemento.
display display: flex; Aplica un diseño flex al contenedor.
width width: 100%; El elemento ocupará el 100% del ancho del contenedor.
height height: 50px; El elemento tendrá una altura de 50px.
border-radius border-radius: 10px; Hace que los bordes del elemento sean redondeados.
box-shadow box-shadow: 5px 5px 10px gray; Añade una sombra gris al elemento.
opacity opacity: 0.5; El elemento se verá con 50% de opacidad.
font-weight font-weight: bold; El texto se verá en negrita.
text-transform text-transform: uppercase; Convierte todo el texto a mayúsculas.
line-height line-height: 1.5; Ajusta la altura de la línea a 1.5 veces la altura del texto.
float float: right; El elemento flotará hacia la derecha.
position position: absolute; top: 870px; left: 850px; Posiciona el elemento 10px desde arriba y 20px desde la izquierda del contenedor.
z-index z-index: 10; Controla el orden de apilamiento del elemento, con un valor mayor sobre otros elementos.
overflow overflow: hidden; Oculta el contenido que excede el área del contenedor.
visibility visibility: hidden; El elemento no será visible pero ocupará espacio.
cursor cursor: pointer; Cambia el cursor a un puntero cuando pasa sobre el elemento.
text-decoration text-decoration: underline; Subraya el texto.
transition transition: all 0.9s ease-in-out; Aplica una transición suave de 0.3 segundos a todos los cambios de estilo.
transform transform: rotate(1deg); Rota el elemento 45 grados.
background-size background-size: cover; Ajusta la imagen de fondo para que cubra todo el contenedor.
